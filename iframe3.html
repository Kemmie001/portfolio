Area Doctor is group project I am participating as the frontend developer. Area Doctor is a medical website to assist patient in communicating with doctors without physical meeting. It also has a dashboard provided after the user or a doctor signs in this was achiced using VUEX for authetication. It is a full VueJS and TailwindCSS project. It has Vue-router used to navigate to the dashbord after the user signs in. For the Backend Laravel was used to create the API. It was deployed on Vercel app It is a work in progress..
