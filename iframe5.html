Description: My To-Do app is a mini App that assist people in organising their day -to map out activities to be done in order of importance. It is a static web app with events like toggle to strike-out activities that has been done, a delete icon to remove actions and it has activities stored in local storage for easy recovery when page is refreshed. It also contains a clear button for a fresh page. It was built with Html5, CSS3, and vanilla Javascript and hosted with Github. View project
